This is a test page to demonstrate theme persistence across navigation.

Navigation Test

When you switch themes and adjust hues on any page, those settings should persist when you navigate between pages. This demonstrates the localStorage functionality working correctly.

Test Instructions:

  1. Change the theme using the theme selector
  2. Check "Change Hue" and adjust the color slider
  3. Navigate back to the home page
  4. Verify your theme and hue settings are preserved

Note: All theme preferences are saved in your browser's local storage and will persist even after closing and reopening the browser.
